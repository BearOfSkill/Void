# Void 2.0
***
Void is a macro mod made for use with [COFL](https://sky.coflnet.com), A hypixel skyblock flipping mod.

# Features
***
- Auto Buy
- Auto Open
- Auto Claim
- Update Checker
- Flip Purchase Webhooks
- Flip Sold Webhooks
- Captcha Alert Webhooks
- Customizable Bed Delay
- Anti-Limbo
- Anti-Lobby
- Auto-Island
